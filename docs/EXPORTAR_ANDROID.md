# Exportar a Android

## Requisitos

- Godot 4.x
- Android Studio o Android SDK
- Plantillas de exportacion Android instaladas en Godot

## Pasos

1. Abre el proyecto en Godot.
2. Ve a **Proyecto > Instalar plantillas de exportacion**.
3. Ve a **Proyecto > Exportar**.
4. Agrega preset **Android**.
5. Configura:
   - Nombre: `Manta: El Astillero`
   - Orientacion: portrait
   - Icono: `icons/icon-192.png` o un icono nuevo de mayor calidad
6. Exporta APK para pruebas.

## Recomendaciones moviles

- Mantener controles grandes.
- Evitar textos pequenos.
- Probar en resolucion vertical.
- Optimizar sprites antes de exportar.
- Agregar sonido y vibracion solo cuando el flujo principal ya sea estable.
