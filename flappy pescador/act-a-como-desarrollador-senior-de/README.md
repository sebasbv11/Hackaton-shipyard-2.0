# Flappy Pescador Manta

Juego 2D en Godot 4 con GDScript. Controlas a un pescador de Manta en su bote y debes esquivar redes de pesca mientras navegas por el Pacifico.

## Abrir en Godot

1. Abre Godot 4.
2. Pulsa **Import**.
3. Selecciona `project.godot`.
4. Abre el proyecto y presiona `F5`.

## Controles

- Celular: tocar pantalla para saltar.
- PC: `Espacio`, flecha arriba o clic para saltar.
- `Esc`: pausa.

## Exportar

- Android: **Project > Export > Android**.
- HTML5/itch.io: **Project > Export > Web**, exporta como `index.html` y sube el `.zip` a itch.io.
